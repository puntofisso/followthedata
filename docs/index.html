/country

/country/{countryname}
It returns the values about a single commodity for a given country.
/country/{countryname}/{commodity}
It returns a list of companies available on the database.
/company
It returns all available information for a single company.
/company/{company name}
It returns values about a given company for a single field of operations (e.g. "downstream").
/company/{company name}/{field}
It returns values about a given company for a single field of operations in a year (e.g. "downstream").
/company/{company name}/{field}/{year}
